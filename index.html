HTML (HyperText Markup Language) es un lenguaje de marcado utilizado para estructurar el contenido de las páginas web. Define elementos como títulos, párrafos, enlaces, imágenes, formularios y otros componentes esenciales para la visualización en navegadores.

Html consiste un una seria de elemento(element) que se utilizan para envolver diferentes partes del contenido para que actúe de una manera determinada
